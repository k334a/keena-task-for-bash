# keena-task-for-bash
Task list ! Minimal color-coded bash script task list for linux. Numbered, and dated by days, hours, minutes, and seconds. 

My first exported project for all the world to use.

## Installation
```bash
curl -sSL https://raw.githubusercontent.com/k334a/keena-task-for-bash/main/install.sh | bash
```

Then restart your terminal or run:
```bash
source ~/.bashrc
```

## Usage
```bash
task "walk the dog" high
task "buy groceries" medium
task list
task done 1
```
